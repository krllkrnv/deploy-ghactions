Быстрый старт

  1. Установите зависимости:
  2. Python: pip install mkdocs
  3. Node: npm ci
  4. Соберите стили PostCSS: npm run styles:docs.
  5. Запустите локальный сервер: mkdocs serve.
  6. Сборка продакшена: mkdocs build (результат — в папке site/).

Структура проекта: - docs/ — Markdown‑контент и ассеты. - custom_theme/ — шаблоны Jinja2 для темы. - src/styles/ — исходники CSS, сборка в docs/assets/styles/index.css.
