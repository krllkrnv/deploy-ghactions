# Руководство пользователя

## Навигация
Меню настраивается в `mkdocs.yml` в секции `nav`. Порядок определяет последовательность в меню и навигацию "Следующая/Предыдущая".

```yaml
nav:
    - Главная: index.md
    - Быстрый старт: getting-started.md
    - Руководство пользователя: user-guide.md
    - Руководство разработчика: developer-guide.md
    - О проекте: about.md
```

## Markdown
- Заголовки: `#`..`######`
- Списки: `-` или `1.`
- Ссылки: `[текст](url)`
- Изображения: `![alt](assets/images/pic.png)`
- Код: тройные бэктики с указанием языка

## Медиа
Изображения размещаются в `docs/assets/images/`, ссылки относительные. При сборке копируются в `site/assets/`.

## Стили и скрипты
- CSS: `extra_css` в `mkdocs.yml`
- JS: `extra_javascript` в `mkdocs.yml`

## Домашняя страница
В `index.md` используется `<section class="home-hero">`. Кастомные классы и стили добавляются в `src/styles/` с последующей сборкой PostCSS.
